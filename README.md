# Landing Page

Minimal customizable landing page scaffold with responsive design and accessibility in mind.

## Quick start

1. Clone the repository
   ```
   git clone https://github.com/arturwyroslak/landing-page.git
   ```
2. Open `index.html` in your browser or host via GitHub Pages / Netlify.

## Structure
- `index.html` - main HTML template with semantic sections
- `styles.css` - variables and minimal styles (mobile-first)
- `script.js` - small helpers and form placeholder

## Customization
- Change colors in `:root` CSS variables in `styles.css`.
- Replace `LOGO` text in header with your image or SVG.
- Update copy in `index.html` for hero, features, and contact.

## Deployment
- GitHub Pages: enable Pages from the `main` branch in repository settings.
- Netlify: drag & drop the repository or connect via Git.
